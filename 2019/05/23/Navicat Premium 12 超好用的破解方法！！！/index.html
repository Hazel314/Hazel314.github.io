2. Navicat Premium 激活

下载激活工具：

网盘地址：Navicat Premium 12 激活工具
提取码：hk4n

下载解压以后是这样的

接下来就是关键步骤了：
在当前目录下打开命令行，比较简单的方法是直接在这个文件夹中选中路径改成cmd，就是这样：

然后用Patch.exe给Navicat的主程序即navicat.exe打个补丁，生成一个私钥。
执行Patch.exe <navicat.exe path>命令，<navicat.exe path>就是navicat.exe安装的完整路径。

其实就是在打开的cmd窗口下输入Patch.exe加空格，然后直接进到navicat安装目录下，把navicat.exe拖进cmd里面，最后帅气回车。
忘记安装路径的话：桌面 - 快捷方式图标 - 右键 - 打开文件所在的位置。

帅气回车之后在当前目录下会生成对应的私钥文件 RegPrivateKey.pem；
然后执行Keygen.exe RegPrivateKey.pem命令，会提示输入用户名和组织名，随便输一下就可以吧，完了回车就会生成一个注册码了。生成注册码以后会要求填入请求码 request code，放着它，现在去找请求码。

拿到注册码就可以开始激活步骤了。
这时候注意要断一下网络，直到激活成功。

打开软件，注册：

输入刚刚获取到的注册码，选择激活：

手动激活：


敲黑板！！！
把弹框里面的请求码 复制粘贴到 刚刚命令行里问你要请求码的地方，回车回车！
生成激活码 License，把生成的License复制粘贴到手动激活弹框里面，激活！！！（Invalid的话直接跳转至最后）

成功！！！

注意：提示软件升级的话尽量不要升级，升级后有概率会出现激活失败

3. 异常情况

生成激活码的时候提示“Not a valid request code”，应该是之前卸载Navicat的时候没有删干净，重新卸载安装一遍，
步骤：卸载、删除文件夹、删除注册表文件。

2. 删除文件夹

3. 删除注册表

win+R 输入 “regedit”，确定

在里面找到 HKEY_CURRENT_USER\Software\PremiumSoft，路径也可以直接复制粘贴转到的
把这个路径下和 Navicat 相关的注册表全部删除，我截图的时候有两个，之前删的时候有三个，不晓得咋肥四
